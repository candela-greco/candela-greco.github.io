Acerca de mi

Mi nombre es Candela, tengo 26 años y desde muy joven siento mucha curiosidad por la tecnología. Siempre fui de las personas que al tener un problema relacionado a lo tecnológico, buscaba respuestas hasta solucionarlo. Pero no fue hasta conocer desarrolladores que me di cuenta lo mucho que me apasionaba. Fue entonces cuando decidí que era algo que quería convertir en mi carrera a tiempo completo. Ahora estoy enfocada en encontrar un puesto en el área de IT donde pueda seguir creciendo y contribuyendo de manera significativa.

HTML

Tengo un sólido dominio del lenguaje HTML para la creación de estructuras de páginas web. Puedo crear documentos HTML semánticamente correctos utilizando etiquetas apropiadas y comprendiendo la importancia de la accesibilidad y la usabilidad. Además, estoy familiarizado con el uso de elementos HTML5 y las prácticas recomendadas para SEO. También tengo experiencia en integrar HTML con CSS para desarrollar sitios web visualmente atractivos.

Css

Tengo experiencia sólida en el uso de CSS para el diseño y la estilización de páginas web. Manejo selectores, propiedades y valores, y puedo crear diseños responsivos y atractivos utilizando hojas de estilo en cascada. Estoy familiarizada con conceptos avanzados como Flexbox para crear diseños flexibles y adaptables de manera eficiente. Además, tengo experiencia en el uso de frameworks como Bootstrap, lo que me permite aprovechar sus componentes preestablecidos y su sistema de rejilla para desarrollar interfaces modernas y responsivas de forma ágil.

C++

Tengo un amplio dominio del lenguaje de programación C++. Estoy familiarizado con los conceptos fundamentales como variables, estructuras de control, funciones y punteros. Además, tengo experiencia en el uso de características avanzadas como la programación orientada a objetos, el manejo de memoria dinámica, los templates y el uso de bibliotecas estándar como STL. He desarrollado aplicaciones y algoritmos complejos en C++ y estoy capacitado para resolver problemas de manera eficiente y estructurada en este lenguaje de programación.
